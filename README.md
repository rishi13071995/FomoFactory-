# FomoFactory-
Backend:-NodeJS,  
Frontend:-React JS,   
Databse:-MongoDB  

## Backend Env:- 
MONGODB_CONNECTION_STRING="mongodb"  
STOCK_API_URL="https://localhost:3000"  
STOCK_SECRET_KEY="xxxx"  
API_VERSION="v1"  
PORT=3000  
APP_TOKEN="xasda"  
NODE_ENV="development"  

## Frontend Env:-  
API_URL = "http://localhost:3000"  
API_KEY = "xxxxx"  

## Startup Commands:-  
## Backend:-  
npm ci  
npm start  
## Frontend:-  
npm ci  
npm run dev
